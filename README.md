quadrant-dragging
=================

Click and drag to resize quadrant of positioned divs on the page. Uses jQuery UI and the `.draggable()` function to resize page's elements based on the position dragged to. See the [demo](svmatthews.github.io/quadrant-dragging) with some [awfully chosen photos](http://www.buzzfeed.com/daves4/unexplainable-stock-photos).
